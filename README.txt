Rover Motion Simulation Program

Description:
This program calculates the distance and time required for a rover to move 
from an origin point to a destination point based on user input.

Requirements:
- Python 3.x installed

How to Run:

1. Open Command Prompt
2. Navigate to the folder containing the file:
   cd Desktop   (or your file location)

3. Run the program:
   python rover.py

Inputs Required:
- Origin coordinates (x1, y1)
- Destination coordinates (x2, y2)
- Initial velocity (m/s)
- Acceleration (m/s^2)
- Maximum velocity (m/s)

Output:
- Distance between points
- Time required to reach destination

Example:
Enter origin x1: 0
Enter origin y1: 0
Enter destination x2: 100
Enter destination y2: 40
Enter initial velocity: 2
Enter acceleration: 1
Enter maximum velocity: 10
